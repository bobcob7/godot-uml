# cmd/godot-uml

## Purpose

CLI entrypoint for the godot-uml binary. Provides commands for rendering `.puml` files, validating syntax, and starting the HTTP server.

## Functionality

- `render` command: render `.puml` files to SVG
- `validate` command: check `.puml` files for errors
- `serve` command: start HTTP server with live editor
- `version` command: print version info
