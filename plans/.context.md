# plans

## Purpose

Implementation plans for godot-uml, ordered by dependency. Each plan builds on prior plans and is independently testable.

## Functionality

- Track implementation progress across 13 plans
- Define acceptance criteria and dependencies for each plan
- Plans are ordered: bootstrap → lexer → parser → diagrams → rendering → API → server → CLI
