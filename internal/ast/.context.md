# internal/ast

## Purpose

Defines the abstract syntax tree node types shared across all diagram types. The AST is the central data structure connecting parsing to rendering.

## Functionality

- Diagram root node containing all elements
- Class diagram nodes: ClassDef, InterfaceDef, EnumDef, Field, Method, Relationship
- Sequence diagram nodes: Participant, Message, Fragment
- Common nodes: Note, Skinparam, Package/Namespace
