# internal/server

## Purpose

HTTP server with PlantUML-compatible API endpoints and a minimal live editor. Static assets are embedded in the binary.

## Functionality

- `POST /render` — accept PlantUML text, return SVG
- `GET /svg/{encoded}` — accept PlantUML-encoded URL, return SVG
- `GET /` — serve live editor HTML page
- Editor: textarea + live SVG preview with debounced auto-render

## Files

| File | Description |
|------|-------------|
| `server.go` | HTTP server implementation |

## Subdirectories

| Directory | Description |
|-----------|-------------|
| `static/` | Embedded HTML/CSS/JS for the live editor |
