# internal/layout

## Purpose

Implements the Sugiyama hierarchical layout algorithm for positioning graph nodes. Takes a graph of sized nodes and edges, produces (x, y) positions.

## Functionality

- Four-phase Sugiyama algorithm: cycle removal, layer assignment, crossing minimization, coordinate assignment
- Handles self-loops, multi-edges, disconnected subgraphs
- Configurable padding and margins

## Files

| File | Description |
|------|-------------|
| `layout.go` | Layout engine orchestration and graph types |

## Subdirectories

| Directory | Description |
|-----------|-------------|
| `layering/` | Layer assignment phase |
| `crossing/` | Crossing minimization phase |
| `coordinate/` | Coordinate assignment phase |
