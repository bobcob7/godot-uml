# internal/layout/coordinate

## Purpose

Implements coordinate assignment for the Sugiyama layout algorithm. Converts layer/position assignments into final (x, y) pixel coordinates.

## Functionality

- Assign x/y coordinates to nodes based on layer and position
- Handle node sizing and spacing
