# internal/tools

## Purpose

Tracks Go tool dependencies via a `tools.go` file with a `//go:build tools` tag. Tool versions are pinned in `go.mod`.

## Functionality

- Pin tool versions (golangci-lint, gofumpt) in `go.mod`
- Install tools via `GOBIN=$(GOBIN) go install`

## Files

| File | Description |
|------|-------------|
| `tools.go` | Tool dependency imports (build-tagged) |
