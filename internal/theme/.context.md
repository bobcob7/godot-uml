# internal/theme

## Purpose

Theme definitions and skinparam resolution engine. Resolves visual properties for any diagram element with cascading precedence: diagram skinparam > theme > fallback.

## Functionality

- Define theme palettes (colors, fonts, sizes, borders, spacing)
- Darcula theme as default (JetBrains dark palette)
- Skinparam directive parsing and override resolution
- Property resolution with fallback chain

## Files

| File | Description |
|------|-------------|
| `theme.go` | Theme engine and Darcula theme definition |
