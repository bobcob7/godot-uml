# internal/font

## Purpose

Provides font metrics and embedded TTF fonts for accurate text measurement. Essential for layout accuracy — text dimensions determine node sizes.

## Functionality

- Embed TTF font(s) in the binary (no external files needed)
- `MeasureText(text, fontSize, fontFamily) → (width, height)`
- Multi-line text measurement support

## Files

| File | Description |
|------|-------------|
| `font.go` | Font metrics and text measurement |
