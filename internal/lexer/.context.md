# internal/lexer

## Purpose

Tokenizes PlantUML text into a stream of typed tokens with accurate source positions (line, column). First stage of the parsing pipeline.

## Functionality

- Tokenize PlantUML syntax into typed tokens
- Track source positions for error reporting
- Produce error tokens for unknown/malformed input
- Support class diagram and sequence diagram syntax
