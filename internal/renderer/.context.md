# internal/renderer

## Purpose

Defines the rendering interface and houses format-specific renderers. Currently supports SVG output.

## Functionality

- Render interface: takes a positioned AST and theme, writes output
- SVG-specific rendering via `svg/` subpackage

## Files

| File | Description |
|------|-------------|
| `renderer.go` | Renderer interface definition |

## Subdirectories

| Directory | Description |
|-----------|-------------|
| `svg/` | SVG output renderer using ajstarks/svgo |
