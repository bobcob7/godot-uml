# internal/renderer/svg

## Purpose

SVG-specific rendering implementation using `ajstarks/svgo`. Generates valid SVG 1.1 output for class and sequence diagrams.

## Functionality

- Render class diagrams with compartments, relationships, and notes
- Render sequence diagrams with lifelines, messages, and fragments
- Apply theme styling to all visual elements

| `svg.go` | SVG renderer implementation |
