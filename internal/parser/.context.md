# internal/parser

## Purpose

Recursive descent parser that converts a token stream into an AST. Handles both class and sequence diagram syntax with error recovery.

## Functionality

- Parse token stream into AST nodes
- Error recovery to report multiple issues
- Support class diagram syntax (classes, interfaces, relationships)
- Support sequence diagram syntax (participants, messages, fragments)
