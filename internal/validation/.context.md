# internal/validation

## Purpose

Semantic validation of parsed AST diagrams. Produces clear, human-readable error messages with source positions.

## Functionality

- Validate class diagrams (duplicate names, missing braces, invalid relationships)
- Validate sequence diagrams (unclosed fragments, undeclared participants)
- Error messages include line:col and suggested fixes

|------|-------------|
| `validation.go` | Validation rules and error reporting |
