# internal/encoding

## Purpose

Implements PlantUML URL encoding/decoding using DEFLATE compression and PlantUML's custom base64 alphabet. Used by the HTTP server for URL-based diagram rendering.

## Functionality

- Encode PlantUML text to URL-safe string
- Decode URL-safe string back to PlantUML text
- Compatible with plantuml.com URL format
