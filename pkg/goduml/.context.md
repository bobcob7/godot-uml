# pkg/goduml

## Purpose

Public library API for godot-uml. Provides a simple interface for Go consumers to render PlantUML diagrams to SVG.

## Functionality

- `Render(io.Reader, io.Writer, ...Option) error` — primary API
- `Validate(io.Reader) []Error` — validate without rendering
- `Parse(io.Reader) (*ast.Diagram, []Error)` — parse and return AST
- Options for theme selection, output format, custom skinparams
