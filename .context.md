# godot-uml

## Purpose

A pure Go implementation of PlantUML that parses PlantUML syntax and renders SVG output. Serves as both a Go library and a standalone HTTP server with a live editor. Default theme is Darcula (JetBrains dark palette).

## Functionality

- Parse PlantUML class and sequence diagram syntax
- Render diagrams to SVG with configurable themes
- Provide a public Go library API (`pkg/goduml`)
- Run an HTTP server with live editor for interactive diagram editing
- CLI for rendering and validating `.puml` files
